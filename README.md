# QuizByte — Подготовка к IT-собеседованиям

Мобильное приложение на Android, которое превращает подготовку к техническим собеседованиям в увлекательную игру.

## Быстрый старт

1. Откройте проект в Android Studio
2. Нажмите **Run** (Shift+F10) — приложение запустится на эмуляторе или устройстве
3. Зарегистрируйтесь с любым email и паролем (от 6 символов)

## Учётные записи

| Роль | Email | Пароль | Доступ |
|------|-------|--------|--------|
| Ученик | любой email | от 6 символов | Квизы, собеседования, дуэли, профиль |
| Администратор | `admin@quizbyte.com` | от 6 символов | Всё выше + админ-панель |
| Автор | `author@quizbyte.com` | от 6 символов | Всё как у ученика + бейдж роли |

## Основные экраны

### Главный экран
- **Стрик / XP / Квизов** — статистика вверху
- **Ежедневный челлендж** — новый микс вопросов каждый день
- **Библиотека** — все квизы по темам
- **Собеседование** — режим с таймером и стрессовой атмосферой
- **Дуэль** — PvP-режим (заглушка)
- **Иконка щита** (только админ) — переход в админ-панель

### Прохождение квиза
1. Выберите квиз из библиотеки или главного экрана
2. Отвечайте на вопросы до истечения таймера
3. После каждого ответа — объяснение с подсветкой правильного варианта
4. В конце — итоговый экран с результатом и процентом

### Режим «Собеседование»
1. Нажмите «Собеседование» на главном экране
2. Выберите сложность (Лёгкий / Средний / Сложный) или включите **адаптивную сложность**
3. Адаптивный режим подбирает вопросы на основе ваших прошлых результатов

### Профиль
- Аватар, имя, email, бейдж роли
- XP, стрик дней, уровень
- Статистика: точность, количество ответов, пройденные квизы
- Достижения (разблокируются автоматически)
- Последние результаты
- **Переключение темы**: Системная / Светлая / Тёмная
- Кнопка выхода из аккаунта

### Админ-панель (только для admin@quizbyte.com)
- **Обзор** — количество вопросов, квизов, тем; разбивка по темам
- **Вопросы** — список всех вопросов с фильтром по теме, удаление
- **Квизы** — список всех квизов, удаление
- **Добавить** — создание нового вопроса или квиза через форму

## Контент

- **155 вопросов** по 9 темам:
  Python, Kotlin, JavaScript, SQL, Алгоритмы, ООП, Паттерны, Git, System Design
- **38 квизов** трёх уровней сложности (Лёгкий, Средний, Сложный)
- Многие вопросы содержат фрагменты кода

## Геймификация

| Механика | Описание |
|----------|----------|
| XP | Начисляется за каждый ответ и завершение квиза |
| Стрик | Считает дни подряд с практикой |
| Уровень | Растёт с накоплением XP |
| Достижения | 13 достижений (первый квиз, 100 ответов, 7-дневный стрик и др.) |
| Адаптивность | Система подбирает сложность под уровень пользователя |

## Технологии

- Kotlin + Jetpack Compose
- Material 3 с тёмной и светлой темами
- Room (локальная БД, работает офлайн)
- DataStore (настройки, сессия, стрик)
- Navigation Compose
- ViewModel + StateFlow + Coroutines
